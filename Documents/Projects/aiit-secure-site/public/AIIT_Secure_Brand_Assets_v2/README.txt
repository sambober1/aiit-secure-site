AIIT Secure Brand Assets v2
============================

This package contains the full and updated brand assets for AIIT Secure.

Contents:
---------
- PrimaryLogo.png (original full-color logo)
- IconLogo.png (original icon only)
- PrintLogo.svg / EPS (original vector logos)
- TaglineLockup.png
- Favicon.ico
- All PNG/JPG variants from original kit

New Additions in v2:
--------------------
- AIIT_Secure_Flat.svg / .png
  * Flat (no shine) version of the logo for white/light backgrounds, documents, and print use.
- AIIT_Secure_Flat_Dark.svg / .png
  * Reversed flat (white on black) version of the logo for dark backgrounds or overlays.

Usage Guidelines:
-----------------
- Use **Original (PrimaryLogo, IconLogo, TaglineLockup)** for marketing, website, and slides where gradients/shine are acceptable.
- Use **Flat** for any white/light background contexts to ensure clarity and readability.
- Use **Flat Dark** for any dark background contexts, overlays, or when white text is required.
- Preserve the vector formats (SVG/EPS) for scalable use in design and print.

Versioning:
-----------
This is v2. The original "AIIT_Secure_Essential_Brand_Assets.zip" should be archived for reference.
